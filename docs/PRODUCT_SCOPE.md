# Product Scope

PurePlay v1 defines a strict functional scope.

## Included

- AI-based source separation
- Deterministic profile assembly
- Standalone Windows executable
- Offline processing
- WAV output generation

## Explicitly Excluded

- Creative remixing
- Audio editing
- Tempo modification
- Pitch shifting
- EQ or compression
- Adaptive intelligence
- User accounts
- Cloud processing
- Automatic dependency installation

PurePlay is a controlled, deterministic tool, not a creative platform.
